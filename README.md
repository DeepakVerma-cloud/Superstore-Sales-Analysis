# Superstore Sales Analysis Dashboard

This project showcases interactive Power BI dashboards for a Superstore dataset.  
It provides insights into sales performance, product trends, and shipping efficiency.

## Dashboards Included
1. Sales Analysis Dashboard: Overview of revenue, profit, and sales trends.
2. Product Analysis Dashboard: Analysis of product categories, top-selling items, and inventory performance.
3. Shipping Analysis Dashboard: Evaluation of shipping methods, delivery times, and region-wise logistics.

Tools & Skills
- Power BI
- Data Analysis & Visualization
- Excel (for data preprocessing if needed)
- Insights Extraction & Dashboard Design
